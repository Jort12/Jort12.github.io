LearnNeuralNet — Neural Network Library from Scratch

A pip-installable neural network library built entirely with NumPy — no TensorFlow, no PyTorch. Implements forward propagation, backpropagation, SGD with momentum, and Xavier/He weight initialization from the ground up. Features a clean, Keras-style API with Sequential, Dense, compile, fit, and predict — plus model save/load, training history tracking, and mini-batch support.

Install: pip install LearnNeuralNet

XFC — Explainable Fuzzy Competition

Built AI agents for the Kessler game — an asteroid-defense simulation where ships shoot bullets and drop mines to destroy asteroids while avoiding collisions. Designed a Takagi–Sugeno fuzzy inference system trained with an Actor–Critic reinforcement learning framework. The critic estimates state-value through weighted fuzzy rule activations, while the actor adjusts rule outputs via TD-error gradients — keeping decision-making fully interpretable through human-readable fuzzy rules. The game supports multi-ship scenarios and runs at 1000X+ real-time speed using compiled wheels for RL training.
